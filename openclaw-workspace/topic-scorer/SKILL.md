# SKILL.md - 选题评分器

## 触发词
`评分` `选题评分` `评估选题`

## 功能
输入选题标题 → 输出评分（0-100）+ 评分理由 + 优化建议

## 使用方式
```
评分：[选题标题]
```

## 评分维度

### 1. 时效性（20分）
- 热点话题：15-20分
- 常青话题：10-15分
- 过时话题：0-10分

### 2. 实用性（20分）
- 可立即应用：15-20分
- 需要额外资源：10-15分
- 纯理论：0-10分

### 3. 独特性（20分）
- 新颖角度：15-20分
- 常见角度但有新意：10-15分
- 老生常谈：0-10分

### 4. 受众面（20分）
- 广泛受众（10万+）：15-20分
- 中等受众（1-10万）：10-15分
- 小众受众（<1万）：0-10分

### 5. 情感共鸣（20分）
- 强烈情感触发：15-20分
- 中等共鸣：10-15分
- 冷门话题：0-10分

## 输出格式

```markdown
## 选题评分报告

**选题**：[标题]
**总分**：[XX/100]

### 维度得分
- 时效性：[X/20] - [简评]
- 实用性：[X/20] - [简评]
- 独特性：[X/20] - [简评]
- 受众面：[X/20] - [简评]
- 情感共鸣：[X/20] - [简评]

### 优化建议
1. [建议1]
2. [建议2]
3. [建议3]

### 推荐标题（3个）
1. [优化标题1]
2. [优化标题2]
3. [优化标题3]

### 判定
- ✅ **推荐写作**（总分≥70）
- ⚠️ **需要优化**（50≤总分<70）
- ❌ **建议放弃**（总分<50）
```

## 示例

**输入**：
```
评分：我用OpenClaw搭建了一条内容流水线
```

**输出**：
```markdown
## 选题评分报告

**选题**：我用OpenClaw搭建了一条内容流水线
**总分**：78/100

### 维度得分
- 时效性：16/20 - AI工具当前热点，但不是最新话题
- 实用性：18/20 - 可立即应用，有实操价值
- 独特性：14/20 - 自动化流程有新意，但类似话题较多
- 受众面：16/20 - 内容创作者广泛，但OpenClaw小众
- 情感共鸣：14/20 - 解决痛点，有共鸣但不强烈

### 优化建议
1. 标题加入数字：增加具体性
2. 突出效果：节省多少时间/提升多少效率
3. 加入对比：vs手动操作

### 推荐标题（3个）
1. 我用OpenClaw搭建了一条内容流水线，从选题到草稿箱全自动，效率提升300%
2. 90%的人不知道的AI内容工厂：一条流水线，30分钟搞定一篇公众号文章
3. 告别熬夜写稿：我用AI搭建了一条内容流水线，副业时间从0到181小时

### 判定
✅ **推荐写作**（总分78≥70）
```

---

## 执行规则

1. 收到触发词后，立即按5个维度评分
2. 每个维度给出简评（1-2句话）
3. 提供3个优化建议
4. 生成3个推荐标题（基于万维钢文风）
5. 给出最终判定
