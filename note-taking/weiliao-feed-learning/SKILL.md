# 喂料学习 (Feed Learning)

当用户说"喂料"并给一个URL时，执行完整的喂料学习流程。

## 触发条件

- 用户说"喂料"、"喂料学习"等词，并附带文章链接
- 用户给微信公众号文章链接，隐含喂料意图

## 完整流程

### Step 1: 提取文章内容

使用 `wechat-article-extract` skill 提取微信文章全文：
1. curl 抓取 HTML（需要 User-Agent）
2. Python 解析提取标题和正文

### Step 2: 按模板创建结构化笔记

**文件路径**: `$OBSIDIAN_VAULT_PATH/01-灵感素材库/3-喂料学习/YYYY-MM-DD_短标题.md`
**文件名格式**: 日期_简洁中文标题（无空格，用驼峰或直接连写）

**模板内容**（frontmatter + 7个分析模块）:

```
---
来源: {公众号名/手动喂料}
标题: {文章标题}
阅读数: {如有}
点赞数: {如有}
采集日期: {YYYY-MM-DD}
原文链接: {url}
标签: {标签1, 标签2}
---

# {文章标题}

## 选题角度
为什么这篇能火？（选题切入角度、蹭了什么热点、解决了什么痛点）

## 标题公式
命中哪个爆款模型？拆解标题结构

## 开头模式
金句/场景/数据/问句？前3句话怎么抓住人的？

## 结构节奏
几段式？哪里快（白描跳切）哪里慢（论断展开）？总字数？

## 金句摘录
直接能学的句子（3-5句）

## 可收藏元素
对比表/清单/数据/流程图？哪个值得收藏的点？

## 我的收获
AI总结：这篇对番茄写作有什么启发？哪些写法可以融入下一篇文章？
```

**重要**: "我的收获"部分必须围绕"番茄写作"（用户的公众号写作风格）来写启发，不是泛泛而谈。

### Step 3: 更新喂料索引

**文件路径**: `$OBSIDIAN_VAULT_PATH/01-灵感素材库/3-喂料学习/喂料索引.md`

在对应月份的条目列表末尾追加一行：
```
- YYYY-MM-DD | **{文章标题}** | 来源：{公众号名} | [原文]({url}) | 标签：{标签1、标签2} | 笔记：[{文件名}]({文件名})
```

### Step 4: 报告完成

告知用户已完成，列出笔记路径和索引更新。

## Obsidian Vault 路径

- 环境变量: `OBSIDIAN_VAULT_PATH`（当前值: `D:\cybertomato`）
- 喂料目录: `{vault}/01-灵感素材库/3-喂料学习/`
- 模板文件: `{vault}/01-灵感素材库/3-喂料学习/模板_喂料笔记.md`
- 索引文件: `{vault}/01-灵感素材库/3-喂料学习/喂料索引.md`

## Pitfalls

- Windows环境下用 `D:/Python/python.exe` 而不是 `python3`（后者返回 exit code 49）
- 微信文章提取必须带 User-Agent header，否则会被拒绝
- 笔记文件名不要有空格，用日期_标题格式
- 索引追加时注意月份对应，如果跨月可能需要新增 `## YYYY-MM` 标题
