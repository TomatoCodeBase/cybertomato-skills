# 观猹 — watcha.cn

## 基本信息
- 用途：AI 产品社区，产品评分、猹评、猹馆讨论
- 登录方式：微信扫码 或 手机验证码（需中国大陆手机号）
- 是否需要登录才能操作：是（未登录时 /quest 等页面会跳转首页并弹出登录框）

## 页面结构要点
- SPA 架构，页面切换无刷新
- 首页内容丰富，snapshot 会很大（建议用 compact=true + element 限定范围）
- web_fetch readability 提取器几乎抓不到内容（JS 渲染），必须用浏览器
- 排行榜有 3 个 tab：热议周榜、口碑飙升榜、活跃用户榜，默认显示热议周榜
- 用户主页 URL 格式：`/@<用户ID>`
- 产品页 URL 格式：`/products/<slug>`

## 操作路径

### 查看活跃用户排行
1. 导航到 `/rank`
2. 点击"活跃用户榜" tab（默认是热议周榜）
3. 用户列表包含：排名、用户名、简介、粉丝数、关注数

### 关注用户
1. 从排行榜点进用户主页（或直接导航 `/@<ID>`）
2. "关注 TA" 按钮在用户名下方、统计数据上方
3. 点击后按钮文字变为"已关注"（即时反馈，无需确认）
4. 回 `/quest` 确认进度

### 查看任务中心
1. 导航到 `/quest`（需要已登录）
2. LV1 任务：完善资料、首篇猹评、首篇帖子、关注3个用户
3. 完成后按钮变为"领取奖励"，点击领取瓜子

### 发布猹评
1. 导航到产品页 `/products/<slug>`
2. 在猹评区域找到输入框
3. 输入内容后发布（待验证具体流程）
