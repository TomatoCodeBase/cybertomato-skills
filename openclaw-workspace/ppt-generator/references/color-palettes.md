# Color Palettes Reference

## 小七姐风格（默认）

**特点**：黑白交替 + 橙色强调 + 极简扁平 + 大留白

| 用途 | 颜色 | HEX |
|------|------|-----|
| 深色页背景 | 纯黑 | `#000000` |
| 浅色页背景 | 纯白 | `#FFFFFF` |
| 强调色 | 橙色 | `#E86800` |
| 深色页卡片 | 深灰 | `#181818` |
| 浅色页卡片 | 浅灰 | `#F5F5F5` |
| 深色页正文 | 灰色 | `#A0A0A0` |
| 浅色页正文 | 深灰 | `#606060` |
| 深色页标题 | 白色 | `#FFFFFF` |
| 浅色页标题 | 近黑 | `#202020` |
| 页码/编号 | 橙色 | `#E86800` |

**排版规则**：
- 封面和结尾用黑底，中间页黑白交替
- 页面左上角放编号（01/02/03...），橙色
- 标题下有一条橙色短装饰线（1.5英寸宽，3pt粗）
- 底部右下角放页码，左下角放品牌标识
- 卡片用圆角矩形，无阴影
- 字体：Microsoft YaHei

**字号层级**：
- 封面主标题：52pt
- 页面标题：36pt
- 卡片标题：20-22pt
- 正文：15-17pt
- 描述文字：14-15pt
- 页码：10pt
- 编号：12pt
