# 什么值得买日常任务 (smzdm-daily)

通过 CDP Proxy 操作用户已登录的 Chrome，自动完成什么值得买每日任务获取经验升级。

## 前置条件

- **Chrome CDP** 已启用
- **CDP Proxy** 运行中：`node skills/web-access-original/scripts/check-deps.mjs`
- 用户已在 Chrome 中登录什么值得买

## 每日任务清单与经验上限

| 任务类型 | 操作 | 每日上限 | 经验值 |
|---------|------|---------|--------|
| 好价点值 | 浏览好价列表，点击"值"按钮 | 20个 | 15 |
| 好价收藏 | 浏览好价列表，点击收藏 | 20个 | 40 |
| 文章点赞 | 浏览文章列表，点击点赞 | 24个 | 15 |
| 文章收藏 | 浏览文章列表，点击收藏 | 24个 | 40 |
| 发布文章评论 | 在文章页发表评论 | 6篇 | 50 |
| 百科商品点评 | 在百科页对商品写点评 | 5个 | 100(上限200) |
| **合计** | | | **~260+经验/天** |

## 核心流程

### 1. 检查当前经验

```
navigate → https://zhiyou.smzdm.com/user/
```

截图查看当前等级和经验值。

### 2. 好价点值 + 收藏

```
navigate → https://www.smzdm.com/haojia/  (好价列表)
```

遍历好价卡片，依次点击"值"和收藏按钮。

### 3. 文章点赞 + 收藏

```
navigate → https://www.smzdm.com/article/  (文章列表)
```

遍历文章卡片，依次点击点赞和收藏按钮。

### 4. 发布文章评论

进入具体文章页，找到评论区输入框，填入评论内容后发布。
注意：评论内容需有变化，避免重复内容被过滤。

### 5. 百科商品点评

进入百科商品页，找到点评区域，填写简短点评后提交。

## 经验与等级

- V1 → V2：经验门槛较低，约100经验即可跨级
- 每日完成全部任务约 260+ 经验
- 经验来源以百科点评（100）和文章收藏/好价收藏（各40）为主

## CDP Proxy API 速查

| 操作 | 命令 |
|------|------|
| 列出 tab | `GET /targets` |
| 新建 tab | `GET /new?url=URL` |
| 导航 | `GET /navigate?target=ID&url=URL` |
| 截图 | `GET /screenshot?target=ID&file=path.png` |
| 执行 JS | `POST /eval?target=ID`，body = 纯 JS 文本（text/plain） |
| 点击 | `POST /click?target=ID`，body = CSS选择器 |
| 关闭 tab | `GET /close?target=ID` |

### 平台特征
- 用户中心：`https://zhiyou.smzdm.com/user/`
- 好价列表：`https://www.smzdm.com/haojia/`
- 文章列表：`https://www.smzdm.com/article/`
- 百科：商品百科页面
- 反爬：有频率限制，操作间需适当间隔（1-2秒）
- 登录态：Cookie 携带

### 操作要点
- 点值/收藏：直接在列表页点击，无需进入详情
- 评论：需进入文章详情页，找到评论区
- 百科点评：需进入百科商品页，找到点评输入区
- 每种任务有每日上限，达到后继续操作无经验收益
- 评论内容需变化，完全相同的评论可能被过滤

### 已知陷阱
- 列表页懒加载，滚动触发更多内容加载
- 操作过快可能触发风控
- 评论提交后需等待确认，避免重复提交
- 经验上限是每日重置，第二天可继续

### 实战记录
- 2026-04-04：首次自动化，经验从3涨到99，V1→V2（单日跨两级）
