# SKILL.md - 灵感捕获
> 创建时间：2026-03-23
> 版本: 1.0

---

## 触发词
`灵感` `灵感捕获` `捕获灵感`

## 功能
输入文章链接 → 提取内容 → 评分 → 提取金句 → 生成选题
输出：保存到灵感库 + 生成3个选题 + 保存到选题库
---
## 使用方式
```
灵感 [文章链接]
        或
灵感 https://example.com/article链接
```

**输出**:
```
✅ 灵感捕获完成
📝 灵感分类: XX | 评分: XX/30
📌 已保存到: 01-灵感素材库/1-日常灵感剪报/YYYYMMDD_文章标题.md
📝 3个选题建议:
  1. 选题1
  2. 选题2
  3. 选题3
```

## 动作流程
1. **提取文章内容**
   - 检测链接类型（微信/小红书）
   - 使用 `wechat-to-obsidian.cjs` 提取
   - 获取标题、作者、内容、发布时间

2. **保存到灵感库**
   - 生成Markdown格式（包含YAML）
   - 文件名格式：YYYYMMDD_文章标题.md
   - 保存到：01-灵感素材库/1-日常灵感剪报/
   - YAML属性：
     - 标题: 文章标题
     - 阶段: 灵感
     - 关联参考: 原文链接
     - 发布平台: 来源平台
3. **灵感分类**
   - 调用「灵感分类器」提示词
   - 评估哲学高度、 实操价值、 情感浓度
   - 计算总分（满分30分）
   - 判断优先级
   - 更新YAML：
     - 优先级: [等级]
     - 评分: [X/30]
     - 分类: [AI/SEO/副业/写作/其他]
4. **灵感提取**
   - 调用「灵感提取助手」提示词
   - 提取核心观点（1-3个）
   - 提取金句（3-5句）
   - 提取关键数据
   - 保存到灵感文件的「提取内容」部分
5. **价值判断**
   - 调用「势能Meta」提示词
   - 识别价值冲突点
   - 判断是否有哲学高度
   - 决定是否值得深入挖掘
6. **金句提炼**
   - 调用「金句提炼器」提示词
   - 从长文中提炼1-2句有感染力的金句
   - 保存到：01-灵感素材库/2-爆款素材片段/
   - 标签：#金句 #情绪爆点
7. **自动生成选题**
   - 调用「选题挖掘机」提示词
   - 基于灵感生成1-3个选题（不同切入点）
   - 文件名格式： YYYYMMDD_选题标题.md
   - 保存到：02-选题仓库/待写选题库/
   - YAML属性：
     - 标题: 选题标题
     - 阶段: 选题
     - 关联参考: [[灵感原文]]
     - 发布平台: 公众号
     - 优先级: [继承自灵感分类]
8. **反馈用户**
```
✅ 已保存灵感： 文章标题
✅ 灵感分类: [分类] | 优先级: [等级] | 评分: [X/30]
✅ 已提取: [核心观点数量]个观点, [金句数量]句金句
✅ 已生成选题:
  1. 选题1
  2. 选题2
  3. 选题3
```

## 注意事项
1. **链接类型检测**: 微信/小红书需要不同的提取方式
2. **YAML强制**: 所有保存的文件必须有YAML属性
3. **触发词唯一**: 避免与其他契约冲突
4. **自动保存**: 不需要用户确认, 自动完成所有步骤
